## 📌 Deskripsi Project

Project ini merupakan implementasi rangkaian elektronik berbasis Arduino yang dibuat menggunakan simulasi pada Tinkercad. Tujuan dari project ini adalah untuk memahami cara kerja komponen elektronik, pemrograman mikrokontroler, serta integrasi antara hardware dan software dalam sebuah sistem sederhana.

Dalam project ini, dilakukan perancangan dan simulasi rangkaian menggunakan berbagai komponen seperti sensor, LED, dan Arduino sebagai pengendali utama. Selain itu, program ditulis menggunakan bahasa pemrograman Arduino untuk mengatur logika kerja rangkaian sesuai dengan kebutuhan.

Hasil dari project ini berupa simulasi rangkaian yang dapat dijalankan secara virtual, serta file desain dan kode program yang diunggah ke GitHub sebagai dokumentasi dan bahan pembelajaran.

## 🎯 Tujuan

* Memahami dasar penggunaan Arduino
* Mempelajari cara kerja rangkaian elektronik
* Mengimplementasikan logika program pada sistem berbasis mikrokontroler
* Mendokumentasikan project dalam bentuk repository GitHub
